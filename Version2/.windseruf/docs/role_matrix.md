# Role Matrix

## Human role
- Human Engineer: Owns intent, risk decisions, and final release authority.

- Orchestrator
- System Analyst
- Requirements Reviewer
- Architect
- Architecture Reviewer
- DevOps Engineer
- Backend Engineer
- Frontend Engineer
- Implementation Reviewer
- QA Engineer
- Security Engineer
- AI Evaluator
- SRE Engineer
- Technical Writer
- Release Reviewer
- Release Manager
- Human Engineer (gate owner)

## Separation of concerns
- Delivery execution: implementation + tests + docs by agents.
- Governance: approvals and scope changes by human engineer.
- Coordination: orchestrator changes workflow state and actor routing from profile YAML.
